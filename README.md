# 研表究明
<code>//研表究明，汉字的序顺并不定一能影阅响读，比如当你看完这句话后，才发这现里的字全是乱的</code>

一个用来打乱汉语句子，以产生对非汉语母语者加密，但对汉语母语者来说仍然能阅读的句子的程序。

## 使用方法
./YanBiaoJiuMing \[-i 输入文件路径] \[-o 输出文件路径] \[-s 随机数种子] \[-z 最长打乱长度]
- 输入文件路径不填则为标准输入（以EOF结束）
- 输出文件路径不填则为标准输出
- 随机数种子不填则为当前unix时间戳（纳秒）
- 最长打乱长度不填则为3
